1. Webpack 3

1. eslint-config-airbnb-base
1. scss with postcss

# Installing and Running
1. Install dependencies
```
npm install
```

2. Running in dev mode (webpack-dev-server)

```
npm start
```

3. Build to /dist
```
npm run build
```
